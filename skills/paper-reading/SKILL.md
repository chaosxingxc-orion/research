# Paper Reading Skill

## 核心能力要求

### 1. 快速筛选能力
- 标题摘要快速评估（< 2分钟/篇）
- 相关性判断（高/中/低）
- 决策树应用（纳入/排除/待定）

### 2. 深度阅读能力
- 结构化阅读方法（IMRaD）
- 关键信息提取（研究问题、方法、结果、结论）
- 批判性思维（优势、局限、适用性）

### 3. 笔记和总结能力
- 标准化笔记模板
- 思维导图制作
- 知识图谱构建

### 4. 知识整合能力
- 跨论文概念连接
- 理论框架构建
- 研究脉络梳理

## 使用步骤

### 1. 初步筛选
1. 阅读标题和摘要
2. 评估与研究主题的相关性
3. 做出快速决策（阅读/跳过/稍后阅读）

### 2. 结构化阅读
1. **Abstract**: 获取整体概览
2. **Introduction**: 了解背景和研究问题
3. **Methods**: 理解研究设计和方法
4. **Results**: 关注主要发现
5. **Discussion**: 理解意义和局限

### 3. 批判性分析
1. 评估研究设计的合理性
2. 识别潜在偏见和局限
3. 判断结果的可信度
4. 思考适用性和推广性

### 4. 笔记记录
1. 使用标准化模板
2. 提取关键信息
3. 记录个人见解
4. 标记重要引用

## 最佳实践

1. **批量筛选**: 先快速筛选大量论文，再深度阅读重要论文
2. **主动阅读**: 带着问题阅读，做笔记，提问
3. **交叉引用**: 注意论文之间的引用关系
4. **定期总结**: 定期整理和总结阅读内容
5. **工具辅助**: 使用文献管理工具和笔记软件

## 常见问题

**Q: 如何处理大量论文？**
A: 使用三遍阅读法：快速筛选、重点阅读、深度研读

**Q: 如何提高阅读效率？**
A: 使用标准化模板、专注于关键部分、做结构化笔记

**Q: 如何避免遗忘？**
A: 定期回顾笔记、建立知识体系、与同事讨论
